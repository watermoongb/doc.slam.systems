# Configuration

![Config](Home.jpg)

- **Host Name** Default is machine name, you can change it want you want.


- **Preferences**. To select TimeLine Sync Mode between players. Default is None, if set to Follower player will sync the playback timeline from Source player, for the best experience, 
	Timeline Sync will affect in conditions below: <br/>
	             1: Use DIRECTOR COR to publish content to players.<br/>
	             2: one player is Source Mode and others is Follower Mode.<br/>
				 3: Playing same playlist and same clip between players.<br/>
				 4: Local Network permission must be Allow when player prompt permission request. <br/>
				 6: IGMP setting in Router and switch must be enabled.<br/>
				 7: At present, do not set more than one Source Mode in LAN.<br/>
