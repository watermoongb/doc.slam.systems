# SLAM PLAY COR-HD


OS requirement: macOS Sonoma 14.0 or later .

Hardware requirement:  macMini 2020 and later, with LED display connected or monitor connected. 

Other collaborative software requirement:  DIRECTOR COR ,SOCCOR COR

Other options: 
1. IGMP enabled in Router and Hub . 
2. Use Ethernet cable connection
3. Nova Star Controller 660pro with USB connected.


Disclaimer:
At present, it has not been full tested on MacBooK, MacBook Air, MacBook Pro, MacBook Neo, and it cannot be guaranteed to running performance on the above devices.
Meanwhile considering the special nature of mobile notebooks, it's not recommended to use mobile devices as a software running platform in official event.
