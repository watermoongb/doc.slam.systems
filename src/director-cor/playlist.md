# PLAYLIST Panel

The purple-tinted area below the ASSETS panel, labeled **PLAYLIST**, displays all the play sequences you have created within the current project. This is a master list of all playlists. One playlist is always selected, indicated by a dark blue background (e.g. "PLAY TIME" when selected). Other created lists such as "Animation Player," "Best Player," "Matrix," etc., are visible.

## Usage

- Selecting a different playlist from this list will immediately populate the central **Editor** with that playlist's specific timeline.

- The list supports standard macOS conventions:
  - **Single click** to select
  - **Double click** to rename
  - Use the **+** icon at the bottom left to create new playlists, or delete existing ones

- The bottom search bar lets you quickly filter through a large number of playlists.
