# INSPECTOR

The red-tinted panel on the far right, labeled **INSPECTOR**, provides detailed context-aware information and properties for the currently selected item or the overall selected playlist. It changes dynamically based on your interaction.

## Preview Window

At the top of the **INSPECTOR** is a live preview window. When an item in the central **Editor** is selected, this window shows a frame from that media file or a small thumbnail of it. It can also display a live feed of the playlist's current time (e.g. a "gamma print" test screen).

## Playlist Info and Layout Mode

When the overall playlist is selected (or nothing specific), the **Playlist Properties** and **Playlist Info** sections provide an overview. In **Layout** mode, this panel also works together with the main preview/layout switch to show how items are arranged and will appear when played:

- **Playlist Properties:** Allows you to change the playlist's internal name (e.g. to "Animation Player").
- **Playlist Info:** Displays a summary including:
  - Total number of items (e.g. 31)
  - Total duration of the entire playlist (e.g. 00:10:20)
  - Date and time it was last modified (e.g. 6/3/2026, 5:24 PM)

## Item Properties

When a single item is selected in the **Editor**, the bottom portion of the **INSPECTOR** updates to show detailed info for that content:

- **Name:** The file name of the selected item (e.g. "12_Gammaprint.mp4").
- **File Path:** The absolute system path on your macOS 16 computer where the source media file is located (e.g. `/Users/watermoon/Downloads/Test/Mar6_Media/12_Gammaprint.mp4`). This is essential for verifying linked media or identifying missing files.
