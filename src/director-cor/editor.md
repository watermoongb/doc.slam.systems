# EDITOR

The yellow-tinted central area, labeled **EDITOR**, is where you construct and manage the selected playlist's timeline. This is a spreadsheet-style table with the following columns:

| Column | Description |
|--------|-------------|
| **Start** | The exact time in the playlist when an item is scheduled to begin playing. |
| **Time** | Shows the precise moment or time value related to playback for each item. *This column may show the start time for the next item.* |
| **Content** | The name of the media file or command associated with that schedule block. |
| **Duration** | The total length of time that specific item will play. |

## Interactivity

The table is highly interactive:

- **Drag and drop** media items from the **ASSETS** panel directly into the **EDITOR** to add them to the sequence.

- Once added, you can **click and drag rows** to rearrange the playback order.

- The interface automatically recalculates the start times and durations for all preceding and subsequent items to maintain a continuous, seamless playback stream.
