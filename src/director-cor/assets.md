# ASSETS Panel

The green-tinted area on the left, labeled **ASSETS**, is your media library. This panel acts as a repository for all source files that can be used in your playlists. It displays a list of file names, often including metadata and file types. You can import new files, organize them into folders (folders are shown with disclosure triangles, e.g. "Europepark" and "26_MVM..."), and manage your file inventory from this view.

## Usage

To add media to a playlist, you can simply **drag a file or a folder** directly from the **ASSETS** list into the central **Editor** table.
